# 10과목 정리본

#### 프로그램 실행 여부에 따른 분류

* 정적 테스트 ex) 리뷰, 정적 분석
* 동적 테스트 ex) 화이트 박스 테스트(명세기반), 블랙박스 테스트(구조기반), 경험기반 테스트(블랙박스 테스트에 속함)

#### 테스트 기법에 따른 분류&#x20;

* 화이트박스 테스트
* 블랙박스 테스트

#### 테스트 시각에 따른 분류&#x20;

* 검증
* 확인

#### 테스트 목적에 따른 분류

* 회복
* 안전
* 성능
  * 부하, 강도, 스파이크, 내구성 테스트 등이 있음&#x20;
* 구조
* 회귀
* 병행
