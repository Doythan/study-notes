# Map

#### Map

* 키(Key)와 값(Value)의 쌍으로 이루어진 데이터를 저장하는 자료구조

***

#### Map의 특징

* 키로 빠르게 검색
* 키 중복 불가 → 덮어쓰게 된다
* 값은 중복 가능
* null 사용 여부는 구현체마다 다름

***

#### Map의 주요기능

* put(key, value) : 값 넣기
* get(key) : 값 꺼내기
* containsKey(key) : 키 존재 여부 확인
* remove(key) : 값 삭제
* size() : 데이터 개수&#x20;

***

#### Map 구현체 (Map 자체는 인터페이스)&#x20;

* HashTable
* HashMap
* TreeMap
* LinkedHashMap

***
