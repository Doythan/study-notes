# Data Structures (자료구조)

### **자료구조 (Data Structures)**

> 순수하게 컴퓨터 과학 관점에서 배우는 “데이터를 담고 다루는 방식” 그 자체

* **선형 구조 (Linear Structures)**
  * Array (배열)
  * Linked List (연결 리스트)
  * Stack (스택)
  * Queue (큐)
  * Deque (덱)
* **비선형 구조 (Non-Linear Structures)**
  * Tree (트리)
  * Binary Tree
  * Binary Search Tree
  * Heap (힙)
  * Graph (그래프)
  * Trie (트라이)
* **Hashing (해싱)**
  * Hash Table
  * Hash Function
* **기타**
  * Set
  * Map

***

→ **핵심**: 자료구조는 “형태”와 “원리”를 배우는 것.

* 배열은 메모리가 연속된다.
* 연결 리스트는 노드와 포인터로 이어진다.
* 해시는 해시 함수를 써서 값을 바로 찾는다.\
